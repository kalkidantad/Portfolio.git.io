LIBRARY DATABASE MANAGEMENT SYSTEM

The system is programmed using MS SQL and aims to create a management system for physical library database management. The goal is to make it easier for library staff to access the books they are requested to lend. The system also makes it possible to track the number of users and books available at the moment, including their detailed information and the number of books that haven't been returned. Additionally, the system includes a method for staff to have different authorization on their access of reading, deleting, and inserting data in the database.Libraries that have been dependent on manual work have faced problems like lack of space, improper management and ineffective staff. Creating and using a database management system that includes data processing, storing and networking applications not only reduces the problems library’s has faced but also provides several functions to track activities with in the library and reduce data redundancy.Library automation refers to the conversion of manual library procedures to computerized system, it includes the technique of using software management system to manage library book-keeping functions and user’s service. And since our country has its own language this library management system contains and accepts books with English and Amharic languages

Objective

To develop a computerized library system that has and accepts books in different languages and functions in an efficient and reliable way to handle, manage and keep books while tracking the borrowed books by keeping records of the books and users that borrow these books.
